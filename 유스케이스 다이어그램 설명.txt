사용 사례: 게임 추천 요청

행위자 : 사용자, 시스템, ChatGPT API.
전제 조건 : 사용자가 로그인되어 있고, 기본 설정을 제공했거나 게임 플레이 기록이 있어야 합니다.
주요 흐름 :
사용자가 추천 페이지로 이동합니다.
사용자는 기본 설정(예: 장르, 플랫폼)을 입력하거나 '사용 기록'을 선택합니다.
시스템은 데이터베이스에서 사용자 데이터(선호도, 기록, 평가)를 검색합니다.
시스템은 데이터를 벡터화하고 ChatGPT API에 프롬프트를 보냅니다.
ChatGPT API는 추천 게임 목록을 반환합니다.
시스템은 사용자에게 추천 사항을 표시합니다.
사용자는 게임을 선택하거나 나중을 위해 추천을 저장합니다.
사후 조건 : 추천 사항이 표시되고 사용자 상호작용이 기록됩니다.
예외 :
기본 설정이 유효하지 않은 경우 사용자에게 명확히 하도록 요청합니다.
API가 실패하면 오류를 표시하고 다시 시도하도록 제안합니다.
비기능적 요구 사항 : 응답 시간 < 3초, UI는 직관적이어야 합니다.
001
요약: 회원가입 기능은 사용자가 시스템에 처음 접근할 때 필수적인 기능으로, 사용자의 기본 정보를 입력받아 계정을 생성합니다. 이를 통해 사용자는 이후 로그인 후 개인화된 게임 추천 및 공략법 제공 서비스를 이용할 수 있습니다.
기본 흐름:
사용자가 회원가입을 시작하는 행동(버튼 클릭)으로 프로세스가 시작됩니다.
시스템은 사용자가 입력해야 할 필수 정보를 요구하는 폼을 제공합니다. 문서에 따르면, 사용자의 게임 장르 선호도와 플레이 이력을 기반으로 추천이 이루어지므로, 회원가입 시 선호 게임 장르를 입력받는 것이 적절할 것으로 보입니다.
사용자가 정보를 입력하고 제출하면, 시스템은 입력값을 검증(예: 중복 아이디 확인, 이메일 형식 확인 등)합니다.
검증이 완료되면 계정이 생성되고, 사용자에게 성공 메시지가 표시됩니다.
002
요약: 로그인 기능은 사용자가 시스템에 등록된 계정으로 인증받아 접근하는 과정으로, 이를 통해 사용자는 개인화된 게임 추천, 공략법 제공, 게임 이력 관리 등의 기능을 이용할 수 있습니다.
사용자가 로그인을 시작하는 행동(버튼 클릭)으로 프로세스가 시작됩니다.
시스템은 로그인에 필요한 정보를 입력받을 수 있는 폼(아이디, 비밀번호 입력란)을 제공합니다.
사용자가 자신의 계정 정보를 입력하고 제출하면, 시스템은 해당 정보를 검증합니다(예: 아이디와 비밀번호 일치 여부 확인).
검증이 완료되면 시스템은 로그인 성공을 알리는 메시지를 표시하고, 사용자를 메인 화면(예: 대시보드 또는 추천 게임 목록 화면)으로 이동시킵니다.
003
요약: 게임 추천 받기 기능은 사용자의 데이터를 분석하여 AI가 맞춤형 게임을 추천하는 기능으로, 사용자가 새로운 게임을 발견하고 더 나은 게임 경험을 할 수 있도록 돕습니다.
사용자가 게임 추천을 요청하는 행동(버튼 클릭)으로 프로세스가 시작됩니다.
시스템은 사용자의 데이터를 수집하고 분석합니다. 문서에 따르면, 게임 장르 선호도, 플레이 이력, 만족도 평가 등이 분석 대상입니다.
AI(ChatGPT)를 활용하여 분석된 데이터를 기반으로 사용자에게 적합한 게임 목록을 생성합니다.
추천된 게임 목록을 사용자에게 표시하며, 목록에는 게임명, 장르, 간단한 소개, 평점 등의 정보가 포함됩니다.
004
요약: 공략법 요청하기 기능은 사용자가 게임 플레이에 도움이 필요한 경우, AI를 통해 맞춤형 공략 정보를 제공받을 수 있는 기능으로, 게임 경험을 향상시키는 데 목적이 있습니다.
사용자가 공략법 요청을 시작하는 행동(버튼 클릭)으로 프로세스가 시작됩니다.
시스템은 사용자가 공략법을 원하는 게임을 선택하거나 입력할 수 있는 인터페이스를 제공합니다(예: 드롭다운 목록 또는 검색 기능).
AI(ChatGPT API)를 통해 선택된 게임에 대한 공략법(전략, 팁, 단계별 가이드 등)을 생성합니다. 문서에 따르면 AI가 공략 정보를 제공하는 역할을 하므로, 이 단계에서 AI의 활용이 핵심입니다.
생성된 공략법을 사용자에게 텍스트나 가이드 형식으로 표시합니다.
005
요약: 게임 결과 등록 기능은 사용자가 플레이한 게임의 결과를 시스템에 기록하는 기능으로, 이를 통해 사용자의 플레이 이력을 관리하고 이후 게임 추천 및 만족도 분석에 활용할 수 있도록 합니다.
사용자가 게임 결과 등록을 시작하는 행동(버튼 클릭)으로 프로세스가 시작됩니다.
시스템은 게임 결과를 입력받을 수 있는 폼을 제공합니다. 입력 항목은 문서에 명시되지 않았으나, 시스템의 목적(추천 및 분석)을 고려하여 게임명, 플레이 시간, 완료 여부, 난이도 등이 포함될 수 있습니다.
사용자가 결과를 입력하고 제출하면, 시스템은 이를 검증(예: 필수 항목 확인)합니다.
결과가 저장되고, 사용자에게 성공적으로 저장되었음을 알리는 메시지가 표시됩니다.
